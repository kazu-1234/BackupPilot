# BackupPilot

BackupPilot は、毎月の手動バックアップ作業をまとめて実行するための Windows 向けバックアップ補助アプリです。

## 主な機能

- フォルダまたは単体ファイルをバックアップ対象として保存
- コピー元とバックアップ先を対象ごとに指定
- サイズと更新日時による差分コピー
- すべてをコピーする強制コピー
- 既存ファイルを上書きする/スキップする設定
- 進捗表示、実行ログ
- ライト、ダーク、システムに合わせるテーマ設定
- GitHub Releases によるアップデート確認
